# -ehir-Rehberi-ve-Etkinlik-Takip-Uygulamas-
Bu proje, şehirdeki kültürel ve sanatsal etkinlikleri tek bir platformda toplar, kullanıcıların doğru ve güncel bilgilere kolayca erişmesini sağlar. Kotlin ile geliştirilen uygulama, etkinlik keşfini, katılımı ve paylaşımını kolaylaştırarak, etkinlik organizatörlerinin daha geniş kitlelere ulaşmalarını hedefler.
